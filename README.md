# Mediaminds_Hackathon_Group4

## Setup Instructions

### Prerequisites
- Python 3.11

### Installation Steps

1. Create a virtual environment:
```python
py -3.11 -m venv lg-env
```

2. Activate the virtual environment:

**Windows:**
```
source lg-env/Scripts/activate
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```
